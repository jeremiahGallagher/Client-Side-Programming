Javascript

Javascript is a client-side scripting language that web developers to interact with webpages. It is a critical part of web authoring. It allows web authors to display and manipulate website information in real time. Javascript can be embedded within an HTML document, or be kept on an external document and referenced.

Document Object Model

The Document Object Model (DOM) is the programmatic interface to HTML and XML documents. The DOM allows a developer to access html and xml directly with languages such as javascript. This allows a user to directly manipulate the elements within a webpage without changing the html code.

Ajax

Ajax (Asynchronous Javascript and XML) is a set of Web development techniques using many technologies on the client-side to create asynchronous Web applications. Using Ajax, developers can send and receive data from a website or application without affecting its performance or appearance. Ajax can make these requests and changes without the need to reload the webpage or application.

Jquery

Jquery is a cross-platform Javascript library that is was designed to simplify client-side html scripting. Jquery is free, open source software. The Jquery syntax is designed to make it easy to navigate a document, access a DOM, handle events and develop Ajax apps.
